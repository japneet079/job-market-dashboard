# Data Science Job Market Analytics Dashboard

An interactive dashboard built with Streamlit to analyze global data science salary trends and predict potential earnings using a Linear Regression model.

## Features
- Global Salary Metrics: Average salary, total job counts, and top job titles.
- Trend Analysis: Visualizations of salary by experience level, country, and job title.
- Salary Predictor: A machine learning tool that estimates salary based on job title, experience, company size, and remote work ratio.

## Tech Stack
- Python
- Pandas (Data Manipulation)
- Plotly (Data Visualization)
- Scikit-learn (Machine Learning)
- Streamlit (Web Framework)

## Installation and Setup

1. Clone the repository:
git clone https://github.com/japneet079/job-market-dashboard.git

2. Install the required dependencies:
pip install -r requirements.txt

3. Run the application:
streamlit run app.py

## Dataset
The project uses the ds_salaries.csv dataset, which contains columns for job titles, experience levels, company locations, and salaries in USD.
